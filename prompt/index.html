배구게임 만드는 프롬프트

문석배구를 만들 때 실제로 쓴 말들을
하나의 프롬프트로 정리했어요. 그대로 복사하세요.

1AI를 엽니다클로드(claude.ai)나 챗GPT에 접속해요. 무료로도 됩니다.
2아래 프롬프트를 통째로 붙여넣어요[대괄호] 부분만 원하는 테마로 바꾸면 나만의 게임이 됩니다.
3받은 파일을 더블클릭index.html 파일 하나가 나와요. 더블클릭하면 게임 시작.
피카츄 배구의 재미요소(점프 + 타이밍 강타)를 담은 배구 게임을 HTML 파일 하나로 만들어줘.

게임 규칙:
- 15점 먼저 내면 승리, 14:14면 듀스(2점 차 승리)
- 혼자하기(AI 상대, 쉬움/보통/어려움) + 둘이하기(키보드 나눠 쓰기)
- 조작은 최대한 단순하게: 좌우 이동 + 버튼 하나로 땅에서는 점프, 공중에서는 스파이크
- 공에 닿는 순간 타이밍 맞춰 누르면 강타가 나가는 손맛이 핵심이야

캐릭터와 테마:
- [여기를 바꿔봐 → 예: 카우보이 인형 vs 우주인 장난감, 배경은 별이 흐르는 우주]
- 캐릭터는 이미지 파일 말고 코드(canvas)로 직접 그려줘. 둥글둥글 귀엽게

디자인:
- [원하는 색감 → 예: 파스텔 민트 톤, 크림색 포인트]
- 득점이나 강타 때 파티클이 터지고 화면이 살짝 흔들리는 타격감

기술 조건 (중요):
- 외부 라이브러리, 인터넷 연결 없이 index.html 파일 딱 하나로. 더블클릭하면 바로 실행되게
- 효과음은 음원 파일 없이 웹오디오로 직접 합성해줘
- 폰에서도 할 수 있게 터치 버튼(◀ ▶ 이동, ▲ 점프)을 넣어줘. 가로 화면 기준
- 키보드는 한/영 상태와 상관없이 작동하게

다 만들면 네가 직접 플레이 테스트를 해보고, AI가 자책골을 넣지는 않는지, 공 속도가 적당한지 밸런스까지 확인해서 고쳐줘.

한 번에 안 예뻐도 괜찮아요

해보고 느낀 점을 그냥 말로 하면 AI가 고칩니다. 제가 실제로 했던 말들:

공이 너무 빨라. 랠리가 길게 이어지게 해줘
AI가 자책골을 넣는데? 고쳐줘
(사진 첨부) 이 사진 색감이랑 분위기로 바꿔줘
이기면 다음 레벨로 넘어가는 무한 레벨 만들어줘
지면 니체 명언이 나오게 해줘

포인트는 하나 — 기술 용어가 아니라, 느낀 점을 그대로 말하는 것.
